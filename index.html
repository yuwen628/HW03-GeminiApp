Hugging Face Token 設定

本設定僅在您選擇 FLUX.1 或 Stable Diffusion 系列模型時需要。 雖然部分模型可免費試用，但強烈建議提供您自己的免費 API Token，以避免因為官方公共請求額度受限而連線失敗。

如何免費取得 Hugging Face Token：

  1. 註冊並登入 Hugging Face 官網
  2. 進入 Settings -> Access Tokens
  3. 建立一個新 Token（類型選擇 Read 即可）
